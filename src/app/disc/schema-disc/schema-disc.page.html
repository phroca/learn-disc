Le DISC
Présentation du DISC

Le schéma représente le DISC avec les 4 parties : La Dominance, l'Influence, la Stabilité, la Conformité.

Le modèle DISC permet d'évaluer le profil des personnes selon ces quatre composantes.

Chaque composante possède ses caractéristiques propres et deux quartiers contigus partagent un certain nombre de caractéristiques. Le profil des personnes est toujours distribué selon les quatre composantes. En général, on observe une ou deux composantes (contiguës) principales. Il est assez rare qu’un profil ne soit distribué que sur une seule composante. Les composantes diamétralement opposées sur le disque étant en opposition, il est assez rare qu’un profil soit uniformément réparti sur les quatre composantes.
